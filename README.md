# chord-memory.net

This is my personal website soon to be available at [chord-memory.net](https://www.chord-memory.net).

## Setup

Must have `python3` and `make` installed.

## Local

To generate the static html files and run the website locally, run
```
make run
```
and then view the site in the browser at http://localhost:8000.

## Calibre

chord-memory.net utilizes [Calibre-Web](https://github.com/janeczku/calibre-web) to access progress % for eBooks on Kobo.

See [calibre-web-aws](https://github.com/chord-memory/calibre-web-aws) for more details.
